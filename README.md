# http-install

Descrever aqui como instalar o servidor http, com php e mariadb para executar aplicações sheer

## Dependencias

Listamos aqui as dependencias gerais de configuração para executar o sheer + noc
* curl
* node
* node gulp
* mariadb
* php
* apache 2

### Dependencias php
* php-imagick
* php-curl
* php-gd
* php-xml
* php-pdo
* php-mysql

### Dependencias apache
* mod_rewrite
* mod_ssl
